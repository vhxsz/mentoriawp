Dúvidas frequentes
Como funciona a mentoria online? +
As sessões são realizadas via videoconferência, com duração de 1h30 cada. Você recebe material de apoio e tem acesso ao WhatsApp para dúvidas entre as sessões.
É necessário que os dois participem? +
Sim, a participação de ambos os cônjuges é fundamental. Trabalhamos com o casal como uma unidade.
Qual a diferença entre os planos? +
O ciclo de 4 encontros é ideal para casais que querem começar com um processo mais curto. O plano completo de 12 encontros oferece acompanhamento mais profundo e contínuo.
Há garantia de resultados? +
Oferecemos garantia de satisfação. Se não ficarem satisfeitos após os primeiros encontros, devolvemos o investimento sem questionamentos.

Não trabalhamos soluções rápidas.
